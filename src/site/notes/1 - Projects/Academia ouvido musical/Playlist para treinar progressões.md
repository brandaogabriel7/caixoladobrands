# Playlist para Treinar Progressões

Músicas organizadas por dificuldade para praticar identificação de progressões de acordes. **Não pesquise as progressões antes de tentar** — o objetivo é descobrir de ouvido.

## Como usar

1. Ouça a música inteira uma vez
2. Identifique quantos acordes diferentes tem por seção
3. Descubra a tônica (a nota que "resolve")
4. Identifique a qualidade de cada acorde (maior/menor)
5. Pense em números (I, IV, V, vi) em vez de letras
6. Anote no MuseScore se quiser documentar
