- Template engines (ferramentas de template) permitem que desenvolvedores gerem conteúdo (HTML, XML, PDF) substituindo valores dinâmicos em locais específicos de templates estáticos
- Permitem ferramentas de programação como loops e condicionais para manipular o conteúdo gerado
- Algumas ferramentas populares em Python são
    - Jinja2
    - Mako
    - Ferramenta de templates nativa do Django
